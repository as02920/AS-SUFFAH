# আস-সুফফা ম্যানেজমেন্ট সিস্টেম (As-Suffah Management System)

এই প্রজেক্টটি একটি রিঅ্যাক্ট (React) এবং ফায়ারবেস (Firebase) ভিত্তিক ম্যানেজমেন্ট সিস্টেম। আপনি এটি জিপ ফাইল হিসেবে ডাউনলোড করে আপনার লোকাল কম্পিউটারে চালাতে পারেন।

## কিভাবে চালাবেন (How to Run Locally)

১. **প্রয়োজনীয় সফটওয়্যার:** আপনার কম্পিউটারে [Node.js](https://nodejs.org/) ইনস্টল করা থাকতে হবে।

২. **ডিপেন্ডেন্সি ইনস্টল করুন:** টার্মিনালে প্রজেক্ট ফোল্ডারে গিয়ে নিচের কমান্ডটি লিখুন:
   ```bash
   npm install
   ```

৩. **প্রজেক্ট চালু করুন:** ইনস্টল হয়ে গেলে নিচের কমান্ডটি দিয়ে ডেভেলপমেন্ট সার্ভার চালু করুন:
   ```bash
   npm run dev
   ```
   সার্ভার চালু হলে ব্রাউজারে `http://localhost:3000` এ প্রজেক্টটি দেখতে পাবেন।

## ফায়ারবেস কনফিগারেশন (Firebase Configuration)

এই প্রজেক্টটি ফায়ারবেস ব্যবহার করে। প্রজেক্টের রুট ডিরেক্টরিতে `firebase-applet-config.json` ফাইলে ফায়ারবেস কনফিগারেশন দেওয়া আছে। আপনি যদি নিজের ফায়ারবেস প্রজেক্ট ব্যবহার করতে চান, তবে এই ফাইলের তথ্যগুলো আপনার ফায়ারবেস কনসোল থেকে আপডেট করে নিন।

## প্রধান ফিচারসমূহ (Key Features)

- ডাইনামিক লোডিং স্ক্রিন (এডমিন প্যানেল থেকে পরিবর্তনযোগ্য)
- পরিচালক ও বিনিয়োগ ব্যবস্থাপনা
- ব্যাংক লেনদেন ট্র্যাকিং
- সুপার এডমিন ও এডমিন রোল ভিত্তিক এক্সেস কন্ট্রোল

## টেক স্ট্যাক (Tech Stack)

- **Frontend:** React, Tailwind CSS, Lucide React, Motion
- **Backend/Database:** Firebase Firestore, Firebase Auth
- **Build Tool:** Vite
